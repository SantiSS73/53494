Para instalar el analizador en Visual Studio CODE y utilizarlo, seguir los pasos:

1- Descargar el repositorio como zip y extraerlo en su computadora. 
2- Copiar la carpeta extraida en Visual Studio Code. 
3- Abrir el archivo de texto llamado "input.txt" (dentro de la carpeta del analizador) reemplazar el contenido del archivo por el de alguno de los archivos de prueba para testear el analizador 
4- Abrir una terminal nueva para la carpeta AnalizadorFOR y escribir "node index.js" para analizar las cadenas.

Se muestrara a continuación el análisis léxico y sintáctico, tabla de lexemas y tokens, árbol de derivación y resultado (salida) del código ingresado, junto a su interpretación en JavaScript. Si la cadena ingresada es inválida, en lugar de la ejecucuión se especificará en donde se produjo el error en el input y por que motivo.
